## Activities

### Manage Tasks

1.  See tasks - completed
2.  Create a new tasks - currently not working
3.  Mark Tasks Done - completed
3.  Prioritize Tasks
4.  Delete Tasks - completed
6.  Edit a tasks name 

## Task Data Model

```
    {
        name : string,
        status : boolean
        id : number
    }
```
